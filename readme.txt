時間を指定して起動したいソフトを順番ずつ起動するためのソフト

myStartUp.txt に実行したいソフトを記入する

  , で区切って指定する
秒数 ,ソフトのパス , パラメータ, 作業フォルダ , ウインドウの表示状態を指定


現時点では、５つの項目を指定可能
	なお、以下では５つの項目をオプションと呼ぶ。


例
150  , C:\Program Files\CLCL\CLCL.exe ,           , C:\Program Files\CLCL\, 1
秒数 , ソフトのパス                   , パラメータ, 作業フォルダ          , ウインドウの表示状態を指定


いくつかルールがあるので説明する。
・# の文字が先頭にある場合のみ、その文はメモのための文として、スルーされる（コメント文になる）

・オプション１とオプション２は必ず指定しなければならない。

・文字がカラなら、初期値を使用する。

・半角スペースのみの行があった場合、スルーされる。

・テキストの内容によって、エラーがでた場合、その行のテキストは実行されなくなる。

・オプション２　はファイル名かどうかをチェックし、ファイル名でなければエラーになる。

・myStartUp.txt のファイルが存在しなかった場合、 myStartUp.txt のファイルが新しく自動で作られる（見本品）



「ウインドウの表示状態を指定」について
	（何も記入しない）		普通に起動
	Max		最大化して起動
	Min		最小化して起動
	NotActive		起動するが最前面に出さない
	Hide	非表示( やらないほうが無難 )


	数字でも指定できる
	1		普通に起動
	2		最大化して起動
	3		最小化して起動
	4		起動するが最前面に出さない
	5		非表示( やらないほうが無難 )



「作業フォルダ」について
	作業フォルダ  が空の場合
	ソフトがあるフォルダが自動的に割り当てられる

	よって次の例は、同じになる
	150 , C:\Program Files\CLCL\CLCL.exe
	150 , C:\Program Files\CLCL\CLCL.exe ,  , C:\Program Files\CLCL\ , 1



-------------------------------------------------------
同じフォルダに入っている　DragFilePath.exe　は、
このソフトと、別のソフトである。

このソフトのウインドウにマウスでファイルをドラッグすると、
ファイルのフルパスが分かる。
（このソフトの役割はそれだけ。）


----------------------------------------------------------



===========おまけ要素=============================

環境変数などを%%で囲むとフォルダ名などに使える
%SystemRoot%	→	C:\WINDOWS  と同じ
%userprofile%	→	C:\Documents and Settings\user　と同じ

例  %SystemRoot%\NOTEPAD.EXE  でメモ帳

他にも独自に作った変数が使える
%MyDocuments%	→	マイドキュメント
%Desktop%		→	デスクトップ
%ThisSoftDir%	→	このソフトがあるフォルダ
%DriveName%		→	C:	など	USBメモリなどから使った場合  E:	などになる

例
30, %ThisSoftDir%\..\Mozilla Firefox\firefox.exe		で FireFox


----------------ソフトのアイコン----------------------------------
doc\iconStartUp.ico
という名のアイコンのファイルを置くと、それがこのソフトのアイコンになる


----------------設定-----------------------------------

user フォルダの中にある
option.txt というファイルを使って設定する。


fontSize=14
view=True
icon=True


 = をはさんでいる、左の文字が設定項目で、
 = の右の文字が設定する値になる。


いくつかルールがあるので説明する。
・# の文字が先頭にある場合のみ、その文はメモのための文として、スルーされる（コメント文になる）

・設定項目の行を削除した場合（一行分のこと）、初期値が設定される。
	（fontSize を 初期値にしたいなら、その一行をまるごと削除するか、# を先頭に置いて、コメント文にするとよい）

・fontSize=
	のように右の値がカラなら、初期値が設定される。

・同じ項目を別の行に書いたなら、後に書いた方の設定が適用になる。

・option.txt　のファイルが存在しなかった場合、初期の設定が書かれたこのファイルが新しく作られる。（見本品）

・ True や False で設定する項目の場合、On や Off の文字でも可能。 1 や 0 でも可能になる。（1 が True 扱い）



以下に、それぞれの設定の内容を説明する

fontSize
フォントサイズを設定する。数字のみ指定可能。
この数字によって、ウインドウの幅と高さの初期値が少し修正される


view
最初にウインドウを表示するかどうかの設定。
設定値は二択なので、True や False 、On や Off、 1 や 0 でも可能になる。


icon
タスクトレイにアイコンを表示しておくかどうかの設定。
True や False で指定
False を指定しても、メニューから「アイコンのみにする」を選べば、アイコンは表示される。
view の設定も、icon の設定も False を指定した場合、ウインドウを表示することができなくなる。




windowCloseIsExit
ウインドウを閉じた時に、このソフトを終了するかどうかの設定。
True や False で指定
False の場合、ウインドウを閉じる動作をした時は、最小化するだけになる。


startActive
最初にウインドウを表示したとき、最前面に表示するかどうかの設定。
True や False で指定
False の場合、最前面に表示しない。
最初にウインドウを表示するときのみ影響し、それ以外は何も変化はない。
view の設定を False にを指定しているのなら、何も変化はない。


fontName
フォント名を設定する。
初期値は空白。



file
myStartUp.txt  ではなく、
別のファイルを「実行したいソフトを記入するファイル」として使いたい場合に指定する。
ファイルパスを記入する。

相対パスの場合は、このソフトがあるフォルダが基準のフォルダになる。
なお、前後に " " があるなら、取り除かれるようになっている。
初期値は空白。






初期値は

fontSize=13
view=True
icon=True
windowCloseIsExit=True
startActive=True
fontName=
file=




----------------パラメーターを指定して起動-----------------------------------

ショートカットのファイルや、コマンドランチャーなどを使って、パラメーターを指定した場合。
option.txt に記入する書式と同じ形式で指定することで、設定をすることができる。

fontSize=13
view=False

を項目１つにつき半角スペースで区切って指定する。
例
C:\myProgramFiles\StartUpTxtFileAhk\StartUpTxtFileAhk.exe  fontSize=13  view=False


パラメーターで指定した内容は、
option.txt  で設定した内容より優先される。
